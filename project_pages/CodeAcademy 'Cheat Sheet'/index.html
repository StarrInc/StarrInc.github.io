Table Tags

Tag Name Use
<table> Table Contains the elements for all HTML Tables
<thead> Table Head Contains a set of table rows defining the column headers
<tbody> Table Body Encapsulates the rows containing actual table data
<tr> Table Row Container of the Table Rows
<td> Table Data Container for Table Data
<tfoot> Table Foot The rows of a footer in a table

Table Attributes

Attribute Name Use
colspan Column Span Defines how many colums a td element will span
rowspan Column Span Defines how many rows a td element will span

Multimedia Tags

Tag Name Use
<img> Image Displays an image on a webpage
<video> Video Displays a video player on a webpage
<audio> Audio Displays an audio player on a webpage

Common Attributes

Attribute Name Use
src Source Defines a source for an html element
href Hyperlink Ref. The URL of a linked resource
rel Relation Relationship of the target object to the link object

Element Name Common Use

<!DOCTYPE html>

<link href="style.css" type="text/css" rel="stylesheet">
